# プライバシーポリシー

本アプリは、氏名・メールアドレス等の個人情報を収集していません。

ただし、ボイス生成機能の提供のため、 ユーザーが入力したテキストをサードパーティ製の外部APIサーバへ送信しています。

生成された音声データおよびアプリ設定は、ユーザー端末内に保存されます。

本アプリでは、利用状況の把握および品質改善のため、Firebase Analytics
を使用する場合があります。また、クラッシュや不具合の解析のため、Firebase Crashlytics を使用しています。

Firebase Analytics および Firebase Crashlytics
により、アプリの利用状況、クラッシュ情報、端末情報、OS情報、アプリのバージョン、その他不具合解析に必要な情報が
Google LLC に送信される場合があります。これらの情報は、アプリの品質改善、不具合修正、安定性向上のために利用します。

本アプリでは、広告配信は行っていません。
