# イカランプタイマー

## 概要

「イカランプタイマー」はイカランプを確認する癖をつけるためのイカしたAndroidアプリです！

## 詳細

ずんだもんと四国めたんが、設定したテキストを一定間隔で読み上げます。

「イカランプ見て！」・「イカランプ確認！」などのテキストを設定しておけば、イカランプへの意識が薄れることを防ぎ、自然とイカランプを確認する習慣を身につけることができます。

読み上げるテキストは自由に設定できるため、イカランプ以外で意識したいことにも活用できます！
